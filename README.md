# Tamra Cook Book
โปรเจกต์นี้เป็น Web Application สำหรับจดสูตรอาหาร สามารถจดสูตรอาหารของตนเอง และดูสูตรอาหารที่ผู้อื่นบันทึกได้ สามารถคอมเม้นต์ และกดไลค์สูตรอาหารที่ถูกใจได้
## ขั้นตอนการติดตั้ง
1. พิมพ์คำสั่ง
```
cd backend
npm i
```
และ
```
cd myfrontend
npm i
```

## ขั้นตอนการ run
เปิด 2 terminal
1. ฝั่ง backand
```
cd backend
nodemon index.js
```
2. ฝั่ง frontend
```
cd myfrontend
npm run serve
```
## เทคโนโลยี
* Vue.js
* Node.js
* Express.js
* MySQL
* Bulma CSS
